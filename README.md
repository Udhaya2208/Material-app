🏗️ Construction Material Estimator 
A professional, mobile-friendly web application designed for Electricians, Plumbers, and Carpenters. This app allows users to quickly select materials, specify details (like color or length), and generate a PDF estimate instantly.

Built with Python (Flask) for the backend and HTML/CSS/JS for a beautiful, responsive frontend.

🚀 Features
📱 Mobile-First Design: Looks and feels like a real mobile app with a dark mode UI.
⚡ Smart Inputs:
Wires: Shows a dropdown for colors (Red, Blue, Yellow, etc.).
Pipes: Shows a text box for Length (e.g., 10ft).
Others: Shows a generic specification box.
🛒 Shopping Cart: Add items, remove them, and see a running count.
📄 PDF Generation: Instantly downloads a professional PDF material list with your company header.
📂 100% Database: Pre-loaded with hundreds of items for Electrical, Plumbing, and Carpentry.
